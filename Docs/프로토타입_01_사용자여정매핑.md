# 프로토타입 01: 사용자 여정 매핑 및 감정 설계

## 🎯 핵심 원칙

**"기술이 아닌 사람 중심의 설계"**

- 배드민턴 동호회원의 실제 하루를 따라가며 설계
- 각 순간의 감정과 니즈를 우선 파악
- 기술은 그 감정을 해결하는 도구로만 사용
